## Deskripsi Project

Sistem Informasi Pemesanan Tiket merupakan aplikasi berbasis web yang digunakan untuk mengelola data pemesanan tiket. Aplikasi ini dibuat menggunakan bahasa pemrograman PHP Native, MySQL, Bootstrap 5, serta menerapkan konsep MVC (Model View Controller).

---

## Fitur Aplikasi

- ✅ Login Admin
- ✅ Registrasi User
- ✅ Dashboard Admin
- ✅ Tambah Data Tiket
- ✅ Edit Data Tiket
- ✅ Hapus Data Tiket
- ✅ Pencarian Data Tiket
- ✅ Logout
- ✅ Tampilan Responsive

## Cara Menjalankan Project

1. Install XAMPP.
2. Jalankan Apache dan MySQL.
3. Salin project ke folder **htdocs**.
4. Buat database di phpMyAdmin.
5. Import file database (.sql).
6. Sesuaikan konfigurasi database pada file **config/Database.php**.
7. Buka browser dan jalankan:

```
http://localhost/uas_251011700344_pemesanan_tiket2/
```
